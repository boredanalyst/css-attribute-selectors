Introduction to Attribute Selectors

Attribute selectors allow you to style elements based on any attribute they have.

Starts With Selector

The middle list item is styled because its class contains the letter z

The contains selector styles only elements with attribute values containing a specific string of characters.

This is not limited by location. So long as the substring of characters exist in the attribute value, it will get styled.
